Магазин

Магазин, по аналогии с местами обитания животных, реализован в виде таблицы с данными из сервера. Таким образом, предоставляются данные имеющихся в наличии кормов, чтобы при походе в магазин при зоопарке человек сориентировался и спланировал покупку заранее.
